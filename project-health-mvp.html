Case Study

GO HEALTH｜健康生活服務

從模糊的GO HEALTH｜健康生活服務題目，收斂為以認知保健、會員任務、健康點與群組互動為核心的第一階段產品驗證。

1. Context｜專案背景

GO HEALTH 是 HAPPY GO 健康生活服務的第一階段 Web MVP。專案初期從認知保健、健康內容與會員日常互動切入，嘗試將 HAPPY GO 既有點數與會員關係，從消費回饋延伸至健康生活、家庭／朋友互動與長期會員經營場景。

這不是一個一開始就有完整需求文件的產品，而是一個需要在服務定位、使用者情境、內容入口、任務機制與活動導流之間逐步收斂的新事業題目。

2. Challenge｜挑戰

第一階段真正要解的問題，不只是功能如何做，而是要確認 HAPPY GO 會員是否願意在消費回饋之外，開始參與健康生活服務。

會員是否願意接受健康生活服務？

健康議題與點數會員服務之間，需要一個足夠自然、低壓的入口。

健康內容與任務能否降低參與門檻？

內容、任務、點數與遊戲化回饋，需要轉化為會員願意開始的日常互動。

MVP 能否支持下一階段判斷？

第一版產品需要留下足夠的行為數據，支持後續服務方向與商業模式判斷。

3. My Role｜我的角色

  • 協助收斂 GO HEALTH 第一階段服務定位、MVP 範圍與使用者流程。
  • 規劃健康內容入口、每日任務、健康點累積、群組互動與活動導流機制。
  • 與醫療合作單位協作，將專業衛教內容轉化為一般會員可理解、可進入互動流程的內容入口。
  • 撰寫功能規格、流程文件與測試規劃。
  • 協調工程、營運、品牌、公關、會員、數據與外部單位，推進開發、驗收與公開上線。
  • 在設計與前端資源有限的情境下，運用 AI 工具完成前端雛形與互動原型。
  • 建立輕量內容管理方式，降低高頻內容調整對工程排程的依賴。

4. Product Decisions｜關鍵產品判斷

先用 Web MVP 驗證

第一階段不先做完整 App，而是用 Web MVP 降低開發成本，快速觀察入口、任務與回訪行為。

先從認知保健切入

不一次展開所有健康議題，而是從認知保健與日常互動開始，降低理解成本與溝通風險。

用低門檻任務降低參與門檻

以每日任務、健康點與互動遊戲，讓健康議題從文章閱讀轉成可開始、可完成的行為。

用群組機制測試陪伴行為

透過家庭／朋友群組觀察陪伴、提醒與週期回饋，是否能提高參與與回訪。

用 Google Sheets 做輕量內容管理

在上線前後內容頻繁調整的情境下，讓營運團隊能維護文字、區塊與顯示狀態，降低對工程排程的依賴。

5. Validation Map｜驗證方向

第一階段 GO HEALTH 的重點，不是一次完成完整健康平台，而是透過低門檻健康互動、點數誘因與群組機制，觀察會員是否願意在消費回饋之外，開始參與健康生活服務。

因此，我將驗證重點整理成三個層次：入口是否能被理解、任務是否能被完成、互動是否有機會被延續。

入口理解

觀察會員是否能理解 GO HEALTH 提供什麼、為什麼要進入，以及健康任務與點數回饋之間的關係。

任務參與

觀察會員是否願意開始任務、完成互動、累積健康點，並透過活動或內容入口進一步探索服務。

關係延續

觀察群組、回饋與週期性內容是否有機會讓健康互動從一次性參與，延伸為家庭／朋友之間的陪伴與提醒。

這些觀察會作為下一階段產品方向的判斷依據，包含內容深度、互動機制、群組設計與醫療合作延伸方式。

6. Current Status / Value｜目前成果與價值

  • 完成第一階段產品版本，推進至公開上線與市場驗證階段。
  • 建立健康內容、每日任務、健康點、群組互動與活動導流的基礎服務流程。
  • 協助 HAPPY GO 建立從點數與消費回饋場景延伸至健康生活服務的第一版產品架構。
  • 建立輕量內容管理流程，提升活動期間內容調整與後續營運彈性。
  • 後續將依會員參與、群組互動、健康內容點擊與回訪數據，判斷 Phase 2 產品方向與醫療合作深化路徑。

7. Reflection｜專案反思

這個專案讓我重新理解 0→1。過去我更習慣從產品規格、功能流程與交付節奏出發；但 GO HEALTH 讓我意識到，新事業 MVP 的難度不只在於把功能做出來，而是在服務定位還需要被收斂、使用者接受度還需要被驗證、內外部團隊仍在對齊的情境下，先做出一個能讓大家看見方向的第一版產品。

對我來說，第一版產品不需要承載全部願景，但必須放下第一塊可被驗證的磚。它要讓使用者知道可以怎麼開始，也要讓團隊知道接下來該觀察什麼、調整什麼、保留什麼。

這也是我在這個專案中最重要的學習：新事業產品經理不只是整理需求或推進開發，而是要在產品定位、產品機制、會員參與與後續商業假設之間，找到一條可以先往前走的路。
