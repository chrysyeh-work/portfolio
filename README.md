# Chrys Yeh｜Product Manager Portfolio

這是一個可直接部署到 GitHub Pages 的靜態個人作品集網站。

## 檔案結構

- `index.html`：首頁
- `about.html`：關於我
- `projects.html`：作品集總覽
- `product-thinking.html`：產品方法論
- `resume.html`：網站版履歷
- `contact.html`：聯絡資訊
- `project-*.html`：專案詳情頁
- `assets/css/style.css`：全站樣式
- `assets/js/script.js`：手機選單、導覽 active、淡入動畫
- `assets/images/`：SVG 插圖
- `assets/files/chrys-resume.pdf`：履歷 PDF

## GitHub Pages 部署

1. 建立 GitHub repository，例如 `chrys-portfolio`
2. 將此資料夾內所有檔案上傳到 repository 根目錄
3. 進入 `Settings → Pages`
4. Source 選擇 `Deploy from a branch`
5. Branch 選擇 `main` / `/root`
6. 儲存後等待部署完成

## 修改方式

- 文案：直接修改各 `.html` 檔案
- 顏色與版型：修改 `assets/css/style.css` 中的 CSS 變數
- 履歷 PDF：替換 `assets/files/chrys-resume.pdf`

## 建議

第一版適合先上線驗證整體定位與作品集敘事。後續可以再加入英文版、更多案例、文章頁或自訂網域。
